== Description ==

Display your WordPress site's information on Panic's Status Board app.

== Installation ==

1. Upload `status-board-for-wp` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why build Status Board for WordPress? =

Why not? Having quick easy access to glanceable information on your site... who doesn't want that?
